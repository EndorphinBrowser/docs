Detailed Description

A class writing a search engine description to an external source.

OpenSearchReader is a class that can be used to write search engine descriptions using the OpenSearch format.

For more information see: http://www.opensearch.org/Specifications/OpenSearch/1.1/Draft_4#OpenSearch_description_document

See also
OpenSearchEngine, OpenSearchReader
